# Pollination Ecology in Quito, Ecuador
This repository contains data and code related to research work about plant-pollinator communities. This project was lead by Daniel Velasco, USFQ alumnus, research associate of the Tropical Biodiversity Institute IBIOTROP, of Universidad San Francisco de Quito USFQ.
Our work describes the structure of the plant–pollinator community in the Valley of Quito, and our first publication focuses on hymenopteran insects: bees and wasps. Manuscript under consideration in the Journal of Urban Ecology.

- A brief guide about the code run for this research can be found <a href="[https://example.com/att-guide](https://davelascoc.github.io/pollinators-quito/Code/Pollinators_2026.html)">here</a>.
